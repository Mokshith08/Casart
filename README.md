## 📖 Project Description

Casart is a responsive e-commerce web application developed as a mini project.  
The main goal of this project is to design and implement a clean, modern, and user-friendly shopping interface.

Users can browse products, like items, add to a cart section , and navigate smoothly across different sections of the website.  
The project emphasizes frontend development, responsive design, and UI consistency.

---

## 🎯 Objectives

- To develop a UI e-commerce website.
- To implement product listing using cards.
- Wishlist section (like).
- Add-to-cart section.
- To ensure compatibility across all devices.
- To enhance practical knowledge of web development.

---

## ✨ Features

- 🔐 Login and Registration pages
- 🏬 Shop page with product cards
- ❤️ Wishlist (Like)
- 🛒 Cart feature 
- 🧭 Sticky navigation bar
- 📱 Fully responsive design
- 🦶 Structured footer section

## ⚙️ How to Run the Project
1. Clone or download the repository:
   `git clone https://github.com/Mokshith08/Casart.git`.

2. Open the project directory:
`cd casart`.

3. Run the project by:
  - Opening `index.html` in a web browser.
  - OR using Live Server in Visual Studio Code.
